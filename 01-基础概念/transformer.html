Multi-Head Attention：多角度关注

每个头独立做 Attention，关注不同的模式。拼接后通过线性层融合回 d_model 维。

Causal Mask：不能偷看未来

Mask 矩阵（下三角）
每个位置能看到的范围
1处理 天 时：只看 [天] → 预测 "问"
2处理 问 时：看 [天, 问] → 预测 "一"
3处理 一 时：看 [天, 问, 一] → 预测 "号"
4处理 号 时：看 [天, 问, 一, 号] → 预测 "成"

实现方式：在 attention score 矩阵的右上角填 -∞，经过 softmax 后变成 0，等于完全看不到。

这就是注意力可视化中右上角全是 0.00 的原因。

mask = torch.tril(torch.ones(seq_len, seq_len))

Positional Encoding：位置信息

为什么需要？

Transformer 一次看到所有字，没有顺序概念：

"我爱你" 和 "你爱我"
→ 不加位置编码，对模型完全一样！

用 sin/cos 生成固定编码，每个位置唯一：

PE(pos, 2i) = sin(pos / 100002i/d)
PE(pos, 2i+1) = cos(pos / 100002i/d)
可视化

X轴=维度，Y轴=位置，颜色=编码值

训练过程

Loss 曲线

每步做三件事

1前向传播：输入 → 模型 → 预测概率
2算 Loss：和正确答案比，计算交叉熵
3反向传播：计算梯度 → 更新参数

Loss 直觉

Loss = 3.99≈ 随机猜（1/54）
Loss = 3.43比瞎猜好一点
Loss = 0.8244% 置信度
Loss = 0.1090% 置信度

Loss = -log(P)，P 是对正确答案的预测概率
